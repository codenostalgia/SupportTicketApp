This App was Developed as a part of NetSkope Recruitment Exam

--------------------------------------------------------

This App is Developed using -
Backend: FastAPI
Frontend: React JS
Database: File based

---------------------------------------------------------

Before running app please ensure, 
you have installed required dependencies like -

Frontend-
├── @testing-library/jest-dom@5.17.0
├── @testing-library/react@13.4.0
├── @testing-library/user-event@13.5.0
├── axios@1.7.4
├── bootstrap@5.3.3
├── react-dom@18.3.1
├── react-scripts@5.0.1
├── react@18.3.1
└── web-vitals@2.1.4

Backend-
├── FastAPI
├── http
├── json
├── pickle
 

you can use-
npm install dependency_name
pip install dependency_name

---------------------------------------------------------
To run the App:

1) Run Backend 
    - navigate to directory "support-tickets-backend"
    - run command "uvicorn.exe main:app --reload"

2) Run Frontend
    -  navigate to directory "support-tickets-frontend"
    -  run command "npm install"
    -  run command "npm start"


---------------------------------------------------------

You can access the App at following default Ports-

Frontend - http://localhost:3000/
Backend - http://localhost:8000/
